template<typename T , typename TMixer = basic_mixer<T>, typename TStorage = basic_storage<>>
size_type leekpp::basic_bloom_filter< T, TMixer, TStorage >::count ( const value_type &  x) const
inline

Test for the likely presence of x in this filter instance.

Keep in mind that a Bloom filter might erroneously test positively for presence when x was never inserted due to false positives. However, this function will never return 0 for a value that was actually inserted (no false negatives).

Returns
0 if the value is not present in this filter; 1 if it looks like the value is present.
